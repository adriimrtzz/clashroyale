ACERCA DE

Este sitio web ha sido desarrollado como parte de un proyecto académico del ciclo de Desarrollo de Aplicaciones Multiplataforma (1º DAM).
Su objetivo es aplicar y reforzar los conocimientos adquiridos en desarrollo web, incluyendo HTML y CSS a través de la creación de una página inspirada en el universo del videojuego Clash Royale.
La página busca combinar diseño, funcionalidad y aprendizaje práctico, mostrando cómo los conceptos teóricos pueden transformarse en una experiencia web interactiva.
Este proyecto no tiene fines comerciales ni está afiliado con Supercell; se trata únicamente de un ejercicio educativo para mejorar mis habilidades como desarrollador en formación.
